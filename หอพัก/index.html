เป้าหมายของเว็บเพจหน้านี้

จ๋อมเว็บเพจ คือ เว็บเพจตัวอย่างที่เป็นผลจากการออกแบบเว็บเพจแบบสแตติกโดยใช้ inline css และ internal css ให้รองรับการแสดงผลกับอุปกรณ์ที่หลากหลายได้เหมาะสม (responsive) ปรับเว็บเพจให้ส่งเข้าประเมินจาก page speed แล้วได้คะแนนสูง และอัพโหลดไปยังฟรีเว็บโฮสติ้งต่าง ๆ แล้วสามารถเรียกใช้โดยไม่ต้องปรับแต่งเพิ่มเติม
Free Cloud Storage ของ Google Drive และ One Drive มีผู้ใช้บริการจำนวนมาก ช่วงหนึ่ง Google Drive เคยให้บริการวาง Host แต่ก็ยุติบริการเมื่อ 31 สิงหาคม 2016 หลังเปิดบริการ 31 สิงหาคม 2015 ไปแล้ว (1 ปีเท่านั้น) ต่อมามีผู้พัฒนาเว็บไซต์ drv.tw (Drive To Web) ให้บริการเชื่อม Drive นำข้อมูลออกมาสร้าง Webpage ทำให้ File แบบ HTML, JS, CSS, TTF ที่วางไว้ใน Folder ต่าง ๆ ของ Drive สามารถถูกเรียกมาแสดงเป็น Webpage บน Browser ผ่าน URL ของ drv.tw แล้วใช้ชื่อ Folder และ File name ที่เคยสร้างใน Drive ได้ เช่น xxx.drv.tw/folder/file
